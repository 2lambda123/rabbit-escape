Rabbit Escape on Linux

Download it

Run it

On most versions of Linux, you can just double-click on the file you downloaded and it will run.

If that doesn't work, try right-clicking and choosing e.g. "OpenJDK Java 7 Runtime". You may need to mark it as executable first.

If that doesn't work, open a Terminal and type (change "Downloads" to something else if you saved it in a different place):

cd Downloads

Then run Rabbit Escape it by typing:

java -jar rabbit-escape-*.jar

If that doesn't work, you may need to install Java, so read the next section.

Is Java Installed?

It's quite likely you already have java installed. If not, to be able to play Rabbit Escape you need to install Java version 6 or above.

To find out, open a Terminal and type:

java -version

If you see something like java version "1.7.0_75" (plus loads more stuff), then you are all set.

If you see something like No command 'java' found (or similar, possibly with loads more stuff too), java is not installed and you should keep reading.

Install Java on Ubuntu/Debian/Mint

If you use Ubuntu, Debian or Mint, install the openjdk-7-jre package by opening a Terminal and typing:

sudo apt-get install openjdk-7-jre

More info: Java on Ubuntu, Java on Debian.

Install Java on Fedora/RedHat/CentOS

If you use Fedora (and possibly RedHat or CentOS) install the java-1.7.0-openjdk package by opening a Terminal and typing:

su -c "yum install java-1.7.0-openjdk"

More info: Java on Fedora, Java on CentOS.

Install Java on Arch

If you use Arch Linux, install the jre7-openjdk package.

More info: Java on Arch.

Other

If you're on a different Linux distribution, check the documentation, or download Java directly from Oracle by following the instructions on java.com.
